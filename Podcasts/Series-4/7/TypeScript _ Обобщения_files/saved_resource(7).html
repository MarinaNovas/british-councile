                                  Alexander • 4 года назад • edited

                                  "TypeScript является строго типизированным языком". Тут есть интересный момент. На самом деле тип данных переменной можно поменять на любой другой, причём даже в strict mode:

                                  let stringVariable: string = "some string";
                                  console.log(`typeof stringVariable: ${typeof stringVariable}`); // string.
                                  //stringVariable = 33; // Type '33' is not assignable to type 'string'. Ok...
                                  //stringVariable = 33 as string; // Type 'number' cannot be converted to type 'string'. Ok...
                                  stringVariable = 33 as any; // Now stringVariable is number.
                                  console.log(`typeof stringVariable: ${typeof stringVariable}`); // number!
                                  console.log(`stringVariable + 11: ${stringVariable + 11}`); // 44

                                  Так как преобразование всё-таки явное, то типизация всё ещё сильная. Но тем не менее факт возможности замены явно определённого типа переменной на другой тип интересен и это не баг - подтверждено командой разработчиков.
